### 修改登录设置

1.修改登录设置-1

| 编号     | 1. 控制台-访问控制IAM-用户-修改登录设置-1                    |
| -------- | ------------------------------------------------------------ |
| 测试点   | 1. 修改登录设置                                              |
| 预置条件 | 1. 主账户登录云平台                                          |
| 步骤     | 1. 进入控制台-访问控制IAM-用户界面，点击要修改的用户名称链接，进入用户详情页面，点击认证管理，点击修改登录设置，修改登录信息<br>2. 点击确认 |
| 预期结果 | 1. 步骤1中，可以修改登录信息<br>2. 在步骤2中，可以查看用户登录信息已更改 |
