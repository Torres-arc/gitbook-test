### 快照删除

1.快照删除-1

| 编号     | 1. 云主机ECS-快照-删除-1                                     |
| -------- | ------------------------------------------------------------ |
| 测试点   | 1. 删除快照成功                                              |
| 预置条件 | 1. 主账户登录云平台<br />2.当快照状态为可用、错误时，才可以删除 |
| 步骤     | 1. 点击快照列表“删除”按键，弹出二次确认弹窗，点击“确认”按键  |
| 预期结果 | 1. 步骤1中，删除快照成功，该快照的镜像也会被删除             |

2.快照删除-2

| 编号     | 1. 云主机ECS-快照-删除-2                                     |
| -------- | ------------------------------------------------------------ |
| 测试点   | 1. 删除快照成功                                              |
| 预置条件 | 1. 主账户登录云平台<br />2.当快照状态为可用、错误时，才可以删除 |
| 步骤     | 1. 勾选任一个或者多个快照前面的小方框，顶部“删除”按键亮起，点击“删除”按键，并进行“确认”操作 |
| 预期结果 | 1. 步骤1中，删除快照成功，该快照的镜像也会被删除             |
