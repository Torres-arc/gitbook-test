### 省份镜像仓库编辑

1.省份镜像仓库编辑-1

| 编号     | 1. 管理平台Admin-省份镜像仓库-省份镜像仓库编辑-1             |
| -------- | ------------------------------------------------------------ |
| 测试点   | 1. 省份镜像仓库编辑成功                      |
| 预置条件 | 1. 超级管理员登录管理平台                          |

2.省份镜像仓库编辑-2

| 编号     | 1. 管理平台Admin-省份镜像仓库-省份镜像仓库编辑-2             |
| -------- | ------------------------------------------------------------ |
| 测试点   | 1. 省份镜像仓库编辑成功                      |
